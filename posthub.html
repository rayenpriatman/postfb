☁ Cloudinary Gratis · Auto Upload · Auto Hapus
Belum dikonfigurasi

Upload file → dapat URL publik → kirim ke Instagram → file otomatis dihapus. Storage tidak pernah penuh.

☁ → 📸 → 🗑 Alur Auto-Hapus
① Kamu pilih foto/video → langsung upload ke folder posthub_temp di Cloudinary
② URL publik dikirim ke Instagram API → Instagram proses & simpan di server mereka sendiri
③ Setelah Instagram berhasil, file di Cloudinary langsung dihapus via backend — storage kembali 0
⚠ Jika tidak pakai backend: set Auto-expiry 1 jam di Cloudinary (lihat cara di bawah) sebagai backup
Setup Cloudinary (2 menit):
1. Daftar di cloudinary.com → gratis
2. Settings → Upload → Upload presets → Add preset
3. Signing mode: Unsigned, Folder: posthub_temp → Save
4. (Opsional, tanpa backend) → di preset, aktifkan Eager transformations dan set Invalidate, atau pakai Auto-delete after: 1 hour via upload tag
5. Copy Cloud Name + Preset ke form di atas

📱 Perintah via Telegram

Kirim perintah dari Telegram, AI akan generate konten lalu auto-post ke platform. Lihat panel Telegram Bot untuk setup.

Format Perintah Telegram:
/post tips jantung sehat → gambar + caption + post semua platform
/post video tips jantung → video + caption + post semua platform
/post fb tips jantung → gambar + post Facebook saja
/post ig video tips stroke → video + post Instagram saja
/preview tips jantung → generate teks saja, tidak posting
/status → cek status koneksi semua platform

🔗 Backend URL

URL server backend yang menangani API calls ke platform. Bisa pakai Django project PostHub yang sudah dibuat, atau backend Node.js/Flask lain.

Backend menerima POST request ke endpoint:
/post-youtube /post-facebook /post-instagram /post-tiktok /cloudinary-delete
Endpoint /cloudinary-delete dipakai untuk hapus file sementara dari Cloudinary setelah Instagram berhasil. Membutuhkan Cloudinary API Secret di backend (tidak bisa dari browser).
